---
title: 'Solana Hook Anatomy'
---

The hook raw data intended for Solana is represented as a versioned transaction with one or more instructions. Thus, only *non-atomic success-required
hooks* are supported.

To craft a proper versioned transaction, use the guide: [Creating Hook data for
Solana](https://docs.debridge.finance/dln-the-debridge-liquidity-network-protocoldln-dropdown/integration-guidelines/interacting-with-the-api/integrating-debridge-hooks/creating-hook-data-for-solana)
